ওয়েব ডিজাইন কিঃ-

ওয়েব ডিজাইন হচ্ছে একটা ওয়েব সাইটের জন্য ডিজাইন করা। আমরা বিভিন্ন ওয়েব সাইটে যখন ভিজিট করি তখন তাদের সুন্দর সুন্দর ডিজাইন দেখতে পাই। যেমন, তাদের Menu, Logo, Sidebar, Image ইত্যাদি থাকে। এই যে এতো কিছু সুন্দর সুন্দর লুক নিয়ে তারা একেক জায়গায় অবস্থান করছে। এতো কিছু মিলে যে একটা ডিজাইন তৈরি হয়েছে তাই ওয়েব ডিজাইন।

কেনো ওয়েব সাইট দরকার?

ওয়েব সাইট আমাদের অনেক গুরুত্বপূর্ণ ভুমিকা পালন করে। বর্তমান সময়ে যেকোনো ছোট বড় ব্যবসা বা নিজের প্রয়োজনে ওয়েব সাইট ব্যবহার অতি গুরুত্বপূর্ণ। নিচে ওয়েব সাইট কেনো দরকার তার ছোট্ট একটা লিস্ট দেয়া হলো।

  1. মানুষ আপনাকে অনলাইনে খুজতে পারে যেকোনো প্রয়োজনে।
  2. ওয়েবসাইট প্রচারণার জন্য খুব ভালো মাধ্যম।
  3. ওয়েব সাইট আপনার সময় এবং খরচ অনেকটাই কমিয়ে দিবে।
  4. আপনার ছোট ব্যবসায় স্বেচ্ছায় লাভ হবে।
  5. আপনার কোম্পানির পণ্য সম্পর্কে তথ্য সহজে কাস্টমার পাবে।

ওয়েব ডিজাইনে আমরা একটা ওয়েব সাইটের জন্য টেমপ্লেট বানাবো। টেমপ্লেট হচ্ছে ওয়েব সাইটের বাহ্যিক(বাইরের অংশ গুলো) অবয়ব তৈরি করা। যা আমরা ওয়েব সাইটে ভিজিট করলে দেখতে পাই। আমরা যে ডিজাইন তৈরি করবো ওয়েব সাইটের জন্য সেটা হবে স্ট্যাটিক ওয়েব সাইট ডিজাইন। পরবর্তীতে যখন কোনো প্রোগ্রাম(পিএইচপি, পাইথন) দিয়ে সেই ডিজাইন কে ডেভলোপ করবো তখন সেটা হবে ডায়নামিক ডিজাইন।

Static Web Page:

Static Web Page তৈরি করা খুব সহজ। এটা সাধারনত html, css দিয়ে করা হয়। এটা করতে কোনো প্রকার প্রোগ্রামিং ভাষা জানতে হয় না বা ডাটাবেস ডিজাইন করতে হয় না। এবং এর কোড প্রত্যেকটা পেজ এ ফিক্সড থাকে। কোনো প্রকার পরিবর্তন হয় না।
